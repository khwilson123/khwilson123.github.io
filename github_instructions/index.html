Forking seems to have had a problem with it. We’re just going to create a site from scratch.

1. Go to Github and login.

2. In the top right, click the + button, and then “New Repository”.

3. Give your repository the name, username.github.io, i.e., if your username is cierra111, give it the name cierra111.github.io.

4. Make the description “My Website”

5. Click “Create Repository”.

6. Click LICENSE.

7. Copy and paste the following into the big black box:

This work is a work in progress. As such, it is licensed under a

[Creative Commons Attribution-NonCommercial-NoDerivatives 4.0

International License](http://creativecommons.org/licenses/by-nc-nd/4.0/).

Plesae see the link for full terms.

8. Scroll to the bottom and fill in the blanks and click Commit New File:

9. Click the New File (+) button:

10. Name your file index.html

11. Copy and paste the following into the big black box:

<html>

  <head>

    <title>Hello, World!</title>

  </head>

  <body>

    HI!!!!!!!!

  </body>

</html>

12. Fill in the blanks at the bottom and click “Commit new file”.

13. Wait a little while and then fire up a browser and go to username.github.io.
